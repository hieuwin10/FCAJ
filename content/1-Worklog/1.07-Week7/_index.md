### Week 7 Objectives:

* Review Database Concepts.
* Understand and deploy Amazon RDS & Amazon Aurora.
* Learn about other data services like Redshift and Elasticache.
* Practice creating network infrastructure (VPC, Subnets, Security Groups) and deploying an application using RDS.
* Practice connecting, configuring, and migrating data (from MS SQL/Oracle to Aurora MySQL).

### Week 7 Achievements:

* Mastered knowledge about different types of databases on AWS (RDS, Aurora, Redshift, Elasticache).

* Successfully connected and deployed a real-world application with RDS.
* Familiarized with the data migration process from traditional RDBMS (MS SQL, Oracle) to AWS (Aurora MySQL).
* Gained basic experience in diagnosing and troubleshooting issues related to Database Migration.
