### Week 6 Objectives:

* Understand and manage quotas for IAM resources.
* Use AWS KMS for S3 data encryption and centralized key management.
* Monitor and trace all AWS account activities via AWS CloudTrail.
* Use Amazon Athena to query logs and analyze security events.
* Compare and practice Access Keys vs. IAM Roles to apply best practices.
* Use AWS Secrets Manager to manage sensitive application information.
